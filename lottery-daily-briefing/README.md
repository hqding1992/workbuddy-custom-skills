# 🎰 彩票新闻每日简报

> 彩票行业新闻自动采编与标准化简报生成工具

---

## ✨ 功能亮点

- 📡 **官方信源自动采集**：覆盖财政部、民政部、体育总局、中国体彩网、中国福彩网、江苏体彩网、江苏福彩网等 10+ 官方固定栏目
- 📊 **9 大彩种开奖汇总**：一站式采集双色球、大乐透、福彩3D、快乐8 等 9 个彩种的开奖号码和奖池
- 🏷️ **五大类信息分类**：政策动态、各地活动、渠道拓展、最新开奖、重大中奖
- 🔄 **持续活动追踪**：自动追踪进行中的活动和渠道案例，避免信息遗漏
- ⏰ **定时自动执行**：支持 Automation 定时任务，每日自动生成简报
- 📤 **多渠道分发**：支持本地保存、邮箱发送、IMA知识库、企业微信、QQ、微信公众号

---

## 🚀 快速开始

### 手动执行

在对话中输入以下任意关键词即可触发：

```
彩票简报 / 彩票新闻 / 每日简报 / 生成简报
```

### 定时自动执行

通过 WorkBuddy 的 Automation 功能设置每日定时任务：

```yaml
名称: 每日彩票新闻简报
提示词: 使用 lottery-daily-briefing Skill 生成今日彩票新闻简报
调度规则: FREQ=DAILY;BYHOUR=13;BYMINUTE=0
工作区: 你的彩票新闻工作区路径
状态: ACTIVE
```

> 💡 **建议执行时间**：12:00-14:00，此时上午开奖数据已发布，下午开奖尚未开始。

## 📋 简报内容

每日简报包含以下五大类信息：

| 分类 | 说明 | 示例 |
|------|------|------|
| 🏛️ **政策动态** | 监管机构政策、规则变更 | 财政部休市安排、彩票资金管理办法 |
| 🎪 **各地活动** | 公益活动、品牌推广、营销促销 | 江苏体彩赠票活动、南京福彩春季宣推 |
| 🏪 **渠道拓展** | 网点创新、跨界合作、渠道转型 | 主题店开业、无人销售终端 |
| 🎱 **最新开奖** | 9 大彩种开奖号码与奖池 | 双色球第XXX期、大乐透第XXX期 |
| 🏆 **重大中奖** | 全国重大中奖事件 | 某地购彩者喜中一等奖 |

### 覆盖彩种

| 类型 | 彩种 | 开奖频率 |
|------|------|---------|
| 体彩 | 超级大乐透、7位数、7星彩、排列3、排列5 | 大乐透/7星彩：一/三/六；7位数：二/三/五/日；排列3/5：每天 |
| 福彩 | 双色球、福彩3D、快乐8、七乐彩 | 双色球：二/四/日；快乐8/3D：每天；七乐彩：一/三/五 |

## 📤 分发方式

简报生成后（Markdown 格式），可通过以下方式分发：

| 方式 | 说明 | 需要配置 |
|------|------|---------|
| 📁 **本地保存** | 自动保存为 `.md` 文件 | 无需配置 |
| 📧 **邮箱发送** | 作为附件或正文通过邮件发送 | 邮箱 Automation |
| 📱 **IMA 知识库** | 两步法分发到 IMA 知识库 | `scripts/ima_distribute.js`（详见 SKILL.md） |
| 💬 **企业微信** | 推送简报摘要 | WorkBuddy 企业微信集成 |
| 💬 **QQ** | 推送简报摘要 | WorkBuddy QQ 集成 |
| 📰 **微信公众号** | 发布到微信公众号 | `Baoyu Post To Wechat` Skill |
