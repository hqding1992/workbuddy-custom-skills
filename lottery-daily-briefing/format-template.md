# 简报格式模板

> 本文件定义彩票新闻简报的标准输出格式。生成简报时严格按照此模板填充内容，确保每次简报风格统一、信息完整。

---

## 📁 文件规范

| 项目 | 规范 |
|------|------|
| **文件名** | `今日彩票新闻简报_YYYYMMDD.md` |
| **文件编码** | UTF-8 |
| **保存路径** | `{workspace}/`（当前工作区根目录） |
| **时间标注** | 标题下方必须标注：`**（YYYY年MM月DD日12:00至YYYY年MM月DD日12:00）**` |

---

> ⚠️ **时间窗口校验**：仅收录开奖时间在简报截止时间（今日12:00）**之前**已开奖的彩种。开奖时间在12:00之后的（如所有晚间开奖），仅收录**上一期**数据，**本期**放入"开奖预告"。

> - 简报截止时间为今日12:00，所有彩种开奖时间均在21:00之后

## 📏 格式注意事项

| 项目 | 规范 |
|------|------|
| **时间格式** | 统一使用"YYYY年MM月DD日"格式 |
| **金额格式** | 使用中文单位（万元/亿元），数字用阿拉伯数字 |
| **来源标注** | 政策动态：来源名称（原文URL）；活动/渠道：来源名称（发布日期），不展示URL |
| **公众号来源** | 格式为"XX省/市体彩/福彩微信公众号（YYYY年MM月DD日发布）" |
| **分隔线** | 各大类之间用 `---` 分隔 |
| **列表层级** | 用 `-` 标记要点，用 `###` 标记条目标题 |
| **表格** | 中奖详情和开奖预告使用 Markdown 表格 |
| **空行** | 各大类之间保持一个空行 |
