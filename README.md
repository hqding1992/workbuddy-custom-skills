# WorkBuddy 自定义 Skills（翰桥）

本仓库归档翰桥自研的 WorkBuddy 技能，便于备份与分享。

## 包含的 Skill

- `hqding-ima-to-wechat` — 从 IMA 知识库读取笔记，AI 生成公众号文章并自动发布草稿箱
- `lottery-daily-briefing` — 彩票行业新闻每日简报自动采编与分发
- `growth-notes` — 轻量成长笔记：做完任务后自省、记录、改进

## 使用方法

将对应子目录复制到 WorkBuddy 的 `skills/` 目录下即可使用（用户级：`~/.workbuddy/skills/`；项目级：`.workbuddy/skills/`）。
